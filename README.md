# RentRide

Website Startup penyewaan kendaraan berbasis Flask.

## Fitur

- Landing Page
- Login
- Register
- Dashboard User
- Dashboard Admin
- Booking Kendaraan
- History Booking
- Profil User

## Teknologi

- HTML
- CSS
- Bootstrap 5
- JavaScript
- Flask
- SQLite

## Cara Menjalankan

pip install -r requirements.txt

python app.py
